# Installation

0. setup userContent.css
1. `git clone git@github.com:FrndlyFox/userContent.git` somewhere
2. import styles in userContent.css
3. toggle features in about:config

### Example

##### directory structure

```
chrome
├ userContent
│ └ youtube.css
├ userChrome.css
└ userContent.css
```

##### userContent.css

```
@import url("userContent/youtube.css");
```
